🏴 Regulament Mafii

Acest regulament se aplică următoarelor organizații:

• 🟣 Magenta

• 🐞 Buburuzelor

• 🇯🇵 Yakuza

• 💀 Camorra

📖 I. Reguli Generale

1.1 Respectarea regulamentului este obligatorie.

1.2 Toți membrii trebuie să respecte liderii și gradele superioare.

1.3 Abuzul de funcție este interzis.

1.4 Exploatarea bugurilor este strict interzisă.

1.5 Cheat-urile și programele externe sunt interzise.

🔫 II. Lupte

2.1 RDM este interzis.

2.2 VDM este interzis.

2.3 Combat Logging este interzis.

2.4 Revenge Kill este interzis.

⚖️ III. Sancțiuni

3.1 Warn.

3.2 Suspendare.

3.3 Eliminare din mafie.

3.4 Ban temporar sau permanent, în funcție de abatere.
